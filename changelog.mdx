## Recent Releases

Stay informed about new features, improvements, bug fixes, and breaking changes in Mekari. You receive notifications through your dashboard and email subscriptions.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for shared documentation projects, allowing multiple users to edit simultaneously.
- Introduced version history with diff views, so you can compare changes and restore previous versions easily.
- Integrated webhook support for custom notifications on document updates.

## Improvements

- Enhanced search functionality with fuzzy matching and full-text indexing for faster results.
- Optimized performance for large documentation spaces, reducing load times by up to 40%.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["bugfix", "security"]}>

## Bug Fixes

- Fixed authentication token refresh issues that caused session expirations during long editing sessions.
- Resolved export failures for documents with embedded images or complex formatting.
- Patched potential XSS vulnerabilities in user-generated content rendering.

## Security

- Updated all dependencies to latest secure versions.
- Added rate limiting to API endpoints to prevent abuse.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Removed legacy v0 API endpoints; migrate to v1 endpoints at `https://api.example.com/v1`.
- Changed default permission model from public to private workspaces.

## New Features

- Core documentation editor with Markdown and rich text support.
- User management and role-based access control.
- Basic analytics dashboard for document views and edits.

</Update>
