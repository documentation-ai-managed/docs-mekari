## What is Mekari?

Mekari is a powerful documentation platform that helps you organize, manage, and share project documentation with your team. You create structured knowledge bases, collaborate in real-time, and deliver searchable, version-controlled docs to users. Whether you build software, manage projects, or run internal wikis, Mekari streamlines your workflow.

<Callout kind="info">
  Mekari integrates seamlessly with your existing tools like GitHub, Slack, and Jira for automated updates and notifications.
</Callout>

## Quick Navigation Guide

Get up and running quickly with these essential steps. You access core features from the dashboard.

<Steps>
  <Step title="Access Dashboard" icon="layout-dashboard">
    Log in at `https://dashboard.mekari.com`. Review recent docs and activity feeds.
  </Step>
  <Step title="Create First Doc" icon="plus">
    Click "New Document" and choose a template. Add markdown, images, or embeds.
  </Step>
  <Step title="Organize Content" icon="folder">
    Use spaces and folders to group related docs. Set permissions for teams.
  </Step>
  <Step title="Publish & Share" icon="share-2">
    Generate public links or embed in apps. Track views and engagement.
  </Step>
</Steps>

## Integrate with Your API

Connect Mekari to your services using our REST API. Start by authenticating.

<CodeGroup tabs="JavaScript,cURL,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/docs', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```bash
  curl -X GET https://api.example.com/v1/docs \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json"
  ```
  ```python
  import requests
  headers = {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
  }
  response = requests.get('https://api.example.com/v1/docs', headers=headers)
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
